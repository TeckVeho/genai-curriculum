#### Slide 07｜Step 1：入力フォームだけ作る — 指示文

**タイトル：** 【Step 1】入力フォームだけ作る（10分）── コツ①「一度に全部頼まない」

**Codex へのタスク入力（コピーして使用）：**

```
シンプルなWebアプリを作ってください。
日報を入力するフォームです。入力欄は次の4つです：
①氏名（テキスト）
②日付（日付入力）
③本日の業務内容（テキストエリア）
④業務に費やした時間（数字、単位は時間）
送信ボタンを押すとアラートで「送信しました」と表示されるだけでOKです。
シンプルで見やすいデザインにしてください。
```

**操作手順：**
1. chatgpt.com/codex を開く
2. 上記の指示文を入力して送信
3. 実行ログを観察 → Changes タブで確認 → Apply
4. ブラウザでプレビューを開く

> 💡 **講師メモ（ノート欄）：**  
> 「まずはフォームだけです。一覧表示も、きれいなデザインも、今は頼みません。これがコツ①『一度に全部頼まない』です。指示文をチャットにも貼りますので、コピーして使ってください。」

