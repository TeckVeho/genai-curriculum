## 判断フローチャート

```
Q1. 解決したい業務は「ChatGPTとの会話の中で完結する」か？
  ├─ はい → Q2へ
  └─ いいえ（独立した画面・フォームが欲しい）→ 【Codex】

Q2. 毎回ほぼ同じ手順・判断を繰り返すか？
  ├─ はい → 【Skills】
  └─ いいえ（都度異なる入力・判断が必要）→ Q3へ

Q3. データの蓄積・一覧表示・集計が必要か？
  ├─ はい → 【Codex】
  └─ いいえ → 【Skills】
```

## 使い分けの早見表

| 観点 | Skills が向いている | Codex が向いている |
|------|-------------------|-----------------|
| 動作環境 | ChatGPTの会話の中で完結 | 独立したWebアプリ・画面 |
| 業務の性質 | 毎回同じ手順・定型作業 | 都度異なる入力・判断 |
| データ管理 | 蓄積・管理不要 | データを保存・一覧表示したい |
| 利用者 | 自分 or チームへの標準化 | 複数人が別々のタイミングで入力 |
| 代表的な用途 | メール生成・FAQ回答・文書作成 | 入力フォーム・台帳管理・レポート出力 |
