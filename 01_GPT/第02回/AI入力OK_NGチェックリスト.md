# AI入力 OK / NG チェックリスト

> **利用タイミング：** 第02回以降、ChatGPT を使うすべての場面で参照してください。  
> デスクに置いておくか、スマートフォンに保存して、入力前に確認する習慣をつけましょう。

---

## ✅ 入力してOKなもの

| カテゴリ | OK な入力例 | ポイント |
|---------|------------|---------|
| 業務内容（一般化） | 「納品が遅れた場合のお詫びメールを書いて」 | 会社名・顧客名を含まない |
| 架空・置き換え情報 | 「A社に対して〜」「B運送の担当者に〜」 | 固有名詞を記号・仮名に置き換えた |
| 社外公開情報 | 「当社のWebサイトに載っている会社概要を要約して」 | すでに公開されている情報 |
| 一般的な業務知識 | 「運送業の請求書に記載すべき項目を教えて」 | 特定の取引先情報を含まない |
| 文章の改善依頼 | 「以下のメール文をより丁寧にして（社名・金額は伏せた文章）」 | 機密部分を削除・マスク済み |
| アイデア出し | 「新人研修の内容案を5つ出して」 | 特定個人の情報を含まない |
| ルール・法律の確認 | 「2024年改正の物流2法について概要を教えて」 | 公開情報の調査・要約 |

---

## ❌ 入力してNGなもの

| カテゴリ | NG な入力例 | なぜNGか |
|---------|------------|---------|
| 顧客の個人情報 | 顧客の氏名・住所・電話番号・メールアドレス | 個人情報保護法の観点からリスクあり |
| 取引先の機密情報 | 「○○株式会社との契約金額は〜」 | 取引先との守秘義務に違反する可能性 |
| 社内機密 | 社内の給与情報・人事評価・未公開の経営計画 | 情報漏洩リスク |
| 金融・口座情報 | 口座番号・クレジットカード番号 | 絶対禁止 |
| パスワード・認証情報 | システムのID・パスワード | 絶対禁止 |
| 特定個人が識別できる情報 | 「○○さんの勤怠記録を見て〜」 | プライバシー侵害リスク |

---

## ⚠️ 注意が必要なもの（条件付きOK）

| カテゴリ | 条件付きOK の例 | 注意点 |
|---------|---------------|--------|
| 社内文書 | 議事録・報告書の要約 | 社名・人名・金額を削除またはマスクしてから入力 |
| メール文章 | 実際のやり取りを参考に改善依頼 | 送信者・受信者名・金額を「A社」「〇〇円」に置き換える |
| 数値データ | 売上・コスト分析 | 実際の数字を「約〇割増」など相対値に変換する |
| 会議での使用 | リアルタイムで入力・要約 | 参加者の個人情報・機密発言を含まないよう確認 |

## 🔧 安全に使うための設定（第02回で確認済み）

### 学習設定オフの確認手順
1. ChatGPT 画面右上のアイコン → **「設定」**
2. **「データコントロール」** を選択
3. **「全員のためにモデルを改善する」** → **オフ** になっていることを確認

### 一時チャットの使い方
- 新規チャット画面上部の **「一時チャット」** ボタンをクリック
- 会話履歴が保存されず、学習にも使われない
- センシティブな内容を扱う際はこちらを優先的に使用

## 📋 入力前の 3秒チェック

入力する前に以下を確認してください：

- [ ] **実在する顧客・取引先の名前が含まれていないか？**
- [ ] **具体的な金額・口座番号・個人情報が含まれていないか？**
- [ ] **社外秘・部外秘の情報が含まれていないか？**
- [ ] **パスワード・認証情報が含まれていないか？**

→ 1つでも「含まれている」なら、**情報を削除・置き換えてから入力してください。**

---

## 💡 困ったときの置き換えルール

| 実際の情報 | 置き換え方 |
|-----------|-----------|
| ○○株式会社（取引先） | 「A社」「取引先X」 |
| 田中様（担当者名） | 「担当者A」「先方担当者」 |
| 123,456円 | 「約〇万円」または「金額は〇〇円」と任意の値 |
| 03-1234-5678 | 電話番号は記載しない |
| info@abc-company.co.jp | メールアドレスは記載しない |
| 受注番号 RCV-20240310-001 | 「受注番号XXX」などに置き換え |

---

> **★ まとめ：** AIは便利な道具ですが、入力した情報はOpenAIのサーバーに送信されます。「この情報、他人に見られても問題ないか？」と一瞬考えてから入力する習慣が大切です。
