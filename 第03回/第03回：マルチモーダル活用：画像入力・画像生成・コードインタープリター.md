# __第03回：マルチモーダル活用：画像入力・画像生成・コードインタープリター__

---

## __1\. 講義概要__

- __フェーズ:__ 応用
- __所要時間:__ 60分
- __主要機能:__ 画像入力（ビジョン）・画像生成・コードインタープリター
- __形式:__ オンライン

### __狙い__

テキスト入力だけだった ChatGPT の活用を大幅に拡張し、**「見る・作る・計算する」の3つの新しい能力**を業務に活かす。手書きメモや伝票の写真を読み取らせ、デスクに張る案内図を生成させ、Excelデータを自動集計させることで、「文字を打つこと」以外の事務仕事もAIで効率化できると実感してもらう。

### __到達目標__

1. 手書きメモや書類の写真を ChatGPT に読み取らせ、テキスト・表に変換できる
2. 業務用の簡単な図解・案内図を DALL-E 3（画像生成）で作成できる
3. コードインタープリターに Excel/CSV データを渡し、集計・グラフ化を依頼できる
4. 各機能の使いどころと注意点（情報漏洩リスク）を理解している

### __準備物（講師）__

- 講師用 PC（ChatGPT ブラウザ版、GPT-4o 使用）
- デモ用画像素材：
	- 手書きの勤怠・経費メモの写真（架空・自作でOK）
	- 海外から届いた英語インボイスのサンプル画像（架空）
- デモ用 CSV ファイル：架空の「部門別月次経費データ（10行程度）」
- ※ 画像生成・コードインタープリターは **Plus プラン以上**。無料版でのフォールバックを案内済みにすること

### __準備物（受講者）__

- PC（ChatGPT ログイン済み）
- スマートフォン（手書きメモや紙を写真撮影するために使用）
- 手元に1枚、紙（レシート・チラシ・手書きメモ。**個人情報・機密情報が写っていないもの**）

### __【08\-25分】 実践①：画像入力（ビジョン）——ChatGPT に「目」をつける__

- __フェーズ:__ デモ / 実践
- __スライド内容:__
	1. __タイトル:__ 手書きメモ・書類をそのままAIに渡す（画像入力）
	2. __キーメッセージ:__ 写真を撮って貼るだけ。テキスト化・翻訳・分析ができる。
	3. __ビジュアル:__ 手書きメモ（紙）→ スマホで撮影→ ChatGPT に貼付→ 整理されたテキストが出力される流れ図

- __講師スクリプト案:__  
「事務でよくある悩みが、**手書きメモの文字起こし**ですよね。急いで書いた文字、クセが強すぎて読めない……なんてこと、ありますよね。  

**ChatGPT は画像が読めます。** 写真を貼り付けて『これ、何が書いてある？』と聞くだけで、テキストに変換してくれます。  

今日のデモでは3つのシーンを見せます。  
**① 手書きメモのテキスト化** → 経費・勤怠メモの写真を渡して表に整理  
**② 英語インボイスの読み取り** → 海外から来た英語の請求書を渡して内容確認  
**③ 書類の要約** → PDF や書類の写真を渡して要点を箇条書き」

- __デモ/操作手順:__

**① 手書きメモのテキスト化**

1. 入力欄の**クリップアイコン**から手書きメモの写真をアップロード
2. プロンプト入力：

> 「この手書きメモを読み取って、テキストに変換してください。さらに、内容を『日付・品目・金額・備考』の表形式に整理してください。」

3. 結果を確認。変換精度・表形式への整理を見せる

**② 英語インボイスの読み取り**

1. 海外インボイス画像をアップロード
2. プロンプト入力：

> 「この英語の書類を読んで、請求書の発行元・請求先・金額・支払い期限を日本語で教えてください。」

3. 日本語で要点を抽出してくれる様子を見せる

- __注意点（スライドに記載）:__
	- **個人情報（顧客名・電話番号）・社外秘の数字が写った画像はアップロードしない**
	- 読み取り精度は100%ではない。必ず人間の目で検品する
	- スマートフォンのカメラで撮影→ブラウザにドラッグ＆ドロップでもOK

### __【40\-55分】 実践③：コードインタープリター——データを自動集計・グラフ化__

- __フェーズ:__ デモ / 実践
- __スライド内容:__
	1. __タイトル:__ Excelがわからなくても大丈夫——データ分析を日本語で頼む
	2. __キーメッセージ:__ 関数も VLOOKUP も不要。「部門別に合計して」と言うだけ。
	3. __ビジュアル:__ CSV ファイル→ ChatGPT 画面→集計表・グラフの変換フロー図

- __講師スクリプト案:__  
「最後は、皆さんが最も苦手とするかもしれない **Excelの集計・グラフ作業** です。  

コードインタープリターという機能を使うと、ChatGPT が Python というプログラム言語でデータを分析してくれます。難しい言葉は覚えなくて大丈夫です。ポイントは**日本語で指示するだけ**ということです。  

例えば：  
**① 部門別に経費の合計を出して**  
**② 前月比で増加率が高い部門を列挙して**  
**③ 棒グラフで可視化して**  

これを1つのプロンプトで頼むだけで、全部やってくれます。VLOOKUP も SUM 関数も不要です。」

- __デモ/操作手順:__

1. 入力欄のクリップアイコンから**架空の「部門別月次経費データ.csv」**をアップロード
2. プロンプト入力：

> 「このデータを読んで、以下の3点を実行してください。①部門別の経費合計を表に整理してください。②合計金額が多い順にランキングを出してください。③部門別経費の棒グラフを作成してください。日本語で出力してください。」

3. 自動で集計・グラフが生成される様子を投影
4. さらに追加指示：

> 「このデータの特徴や気になる点を、経営者目線で3点コメントしてください。」

5. コメント付きで分析してくれる様子を見せる

- __注意点（スライドに記載）:__
	- **実際の売上データ・個人情報・顧客情報が含まれる CSV は使用しない**
	- 集計結果は必ず人間の目で検品する（稀に計算誤りが起きる場合あり）
	- Plus プラン以上で使用可。無料版でも一部利用可能な場合あるが機能制限あり

- __受講者演習（10分）:__
	- 架空のサンプル CSV（講師から配布またはテキストを画面表示）を使って、上記のいずれかを試す
	- **CSV がない場合の代替：** 手入力でも可  
	  例：「以下のデータを表にして：総務 15万円、営業 32万円、物流 28万円、経理 12万円。棒グラフも作成して。」

## __3\. 想定される質問と回答 \(Q&A\)__

- __Q1: 「手書き文字の読み取り、本当に間違えませんか？」__
	- __A（回答）:__ 「正直に言います。**100%ではありません。** 略語や、あまりに崩れた字は間違えます。ですが、『ゼロから手入力する』より、『AIが作ったものを修正する』方が5倍以上早いです。**『下書きを作ってもらって、修正する』**という使い方が正解です。必ず人間の目で最終確認してください。」

- __Q2: 「写真をアップロードして、情報漏洩しませんか？」__
	- __A（回答）:__ 「良い質問です。**個人情報（顧客名・電話番号・住所）や社外秘の数字が写っているものは絶対に避けてください。** 機密情報が写る部分は付箋や手で隠して撮影するか、ダミーデータで試しましょう。また学習設定のオフ・一時チャットの活用も忘れずに。別紙『AI入力OK_NGチェックリスト』を参照してください。」

- __Q3: 「画像生成で作ったイラスト、そのまま使っていいですか？」__
	- __A（回答）:__ 「**社内資料・研修用途のたたき台として使うのは問題ありません。** ただし、社外に公開したり、商業目的で使ったりする場合は、ChatGPT の利用規約や著作権の確認が必要です。実在の人物・有名キャラクター・商標ロゴは生成できない設計になっています。あくまで『下書き・参考イメージ』として活用してください。」

- __Q4: 「コードインタープリターで集計したデータ、Excel に戻せますか？」__
	- __A（回答）:__ 「はい。ChatGPT が生成した表の内容を**コピー＆ペーストして Excel に貼り付ける**か、『この集計結果を Excel の CSV 形式でダウンロードできますか？』と聞くと、CSV ファイルを出力してくれます。ダウンロードしたファイルを Excel で開けます。」
