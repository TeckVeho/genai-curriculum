# __第05回：カスタムGPT＋プロジェクト＋Skills：自分だけの「業務特化AI」を作る__

---

## __1\. 講義概要__

- __フェーズ:__ 応用
- __所要時間:__ 60分
- __主要機能:__ カスタムGPT（My GPTs）・プロジェクト・Skills
- __形式:__ オンライン

### __狙い__

毎回同じ長い指示を打ち込む手間を省き、**「業務特化の AI アシスタント」を一度作れば何度でも使い回せる**仕組みを習得する。カスタムGPT で「専属業務 AI」を作り、プロジェクト機能で「常時文脈を記憶させ」、Skills でチームに展開する——この3段階で、個人の効率化をチーム全体の資産に昇華させる考え方を体得する。

### __到達目標__

1. カスタムGPT（My GPTs）を1つ作成し、「名前・指示・知識・能力」を設定できる
2. プロジェクト機能で、よく使う情報・指示をコンテキストとして保存できる
3. Skills の概念（チームでの共有・ワークフロー標準化）を理解している
4. 「毎回打つ指示がどの機能で省略できるか」を自分で判断できる

### __準備物（講師）__

- 講師用 PC（ChatGPT Plus 以上）
- デモ用カスタムGPT の設計メモ（名前・Instruction・知識ファイル）
- Skills の画面（Team/Business プランがある場合）または説明スライド

### __準備物（受講者）__

- PC（ChatGPT ログイン済み）
- **カスタムGPT作成：Plus プラン以上が必要。無料版の方はプロジェクト機能のみ体験**
- 「自分がよく使うプロンプト」を1つ思い浮かべておく（宿題の振り返りから）

#### __カスタムGPT 作成デモ（15分）__

**ステップ1：作成画面を開く**

1. ChatGPT の左サイドバー「GPT を探す」→「＋ 作成する」または「My GPTs」
2. 「設定（Configure）」タブを選択（会話ではなく直接設定するタブ）

**ステップ2：基本情報を入力**

- **名前：** `取引先向け・請求連絡メール係`
- **説明：** `物流会社の取引先に送るビジネスメールの下書きを作成するアシスタント`
- **プロフィール画像：** 自動生成または省略

**ステップ3：Instruction（指示）を入力**

```
あなたは物流会社の経験豊富な一般事務担当者です。
ユーザーから「用件」「相手の属性（既存取引先／新規）」「希望内容」を受け取ったら、
丁寧で簡潔なビジネスメールの本文案を作成してください。

ルール：
・です・ます調、結論から書く
・時候の挨拶は書かない
・金額や個別契約に関する断定は避け「要確認」と明記する
・メール件名の候補も2〜3つ提案する
・文字数は200文字程度
・個人情報（実在する顧客名・電話番号）は絶対に使わない
```

**ステップ4：知識ファイルのアップロード（オプション）**

- 「Knowledge（知識）」欄に、自社の文書作成ルール・よく使う定型文 PDF をアップロード可
- 例：「メール文例集.pdf」「社内文書フォーマット.docx」
- **機密情報が含まれる場合は学習設定を確認してからアップロード**

**ステップ5：機能の設定**

- **Web 検索：** オン（最新情報を参照する場合）
- **画像生成：** 今回はオフ
- **コードインタープリター：** オフ

**ステップ6：テスト入力**

> 「用件：請求書の再発行依頼、希望：今週中、相手：既存取引先」

生成されたメールが Instruction のルール通りになっているか確認する

**ステップ7：保存・公開設定**

- 「自分だけ（Only me）」を選択して保存
- 「リンクを知っている人（Link）」にすると同僚に URL を送って使ってもらえる

#### __プロジェクト作成デモ（8分）__

**ステップ1：プロジェクトを作成**

1. ChatGPT の左サイドバー「新しいプロジェクト」
2. 名前を入力：「月次報告書作成」
3. プロジェクトが左サイドバーに表示されることを確認

**ステップ2：カスタム指示を追加**

1. プロジェクト設定（歯車アイコン）を開く
2. 「カスタム指示（Instructions）」欄に入力：

```
このプロジェクトでは月次の業務報告書を作成します。
・読み手は会社の上司（役職：部長）
・文体は丁寧語（です・ます調）
・数値は具体的に（〇〇件、〇〇円 など）
・個人情報・社外秘の数字は入れないこと
```

**ステップ3：ファイルを追加**

1. 「ファイルを追加」から過去の報告書テンプレートや参考 PDF をアップロード
2. 以後のチャットで「このプロジェクトのファイルを参照して」と言えば自動参照

**ステップ4：テスト入力**

> 「今月の経費報告書の骨子を作ってください。」

→ 上記の指示が最初から適用された状態で回答される

## __3\. 想定される質問と回答 \(Q&A\)__

- __Q1: 「カスタムGPT とプロジェクトの使い分けはどうすればいいですか？」__
	- __A（回答）:__ 「カスタムGPT は**特定の業務に特化した専用 AI を作る**場合（例：メール係、日報係）、プロジェクトは**同じテーマで継続的にやりとりするときの文脈を保存する**場合（例：月次報告書プロジェクト、取引先対応プロジェクト）に向いています。どちらもよく使う指示の手打ちを省く機能なので、迷ったらプロジェクトから試してみてください（無料版でも使えるので）。」

- __Q2: 「カスタムGPT に社内資料をアップロードしても大丈夫ですか？」__
	- __A（回答）:__ 「アップロードした資料は、そのカスタムGPT の知識として使われます。機密情報・個人情報が含まれるファイルのアップロードは、**学習設定がオフになっているか、Business/Team プランの企業契約かどうかを確認してから**行ってください。公開されている社内規程や、自社が許可した手順書・フォーマット等からスタートすることをおすすめします。」

- __Q3: 「カスタムGPT を同僚にも使ってもらいたいのですが、どうすれば？」__
	- __A（回答）:__ 「公開設定を**『リンクを知っている人（Link）』**にすることで、URL を共有するだけで同僚も使えます。**カスタムGPT の使用（閲覧・利用）は無料版でも可能**なので、Plus プランの方が作成して URL を配布する運用がおすすめです。チーム全員が Plus プランなら、Skills でより体系的に共有できます。」

- __Q4: 「プロジェクトに入れたファイルは削除できますか？」__
	- __A（回答）:__ 「はい。プロジェクト設定から、いつでもファイルの追加・削除・更新が可能です。古くなった手順書は削除して、最新版に差し替えるとより精度の高い回答が得られます。」
