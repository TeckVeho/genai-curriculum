## 1. 導入：AIチームで仕事が終わる時代

- AIチームを使うことで、**2ヶ月かかっていた仕事が2時間**で終わるケースも出てきている
- 企業への AI 導入支援を通じて感じるのは、AI が経営・採用に与える影響力の大きさ
  - 資料作成が1日→1時間になるような業務効率化はもちろん、AI が「人のように」仕事をこなせるようになったことで、**新卒採用人数を絞る動き**が企業側に出てきている
  - 実際に日本の大手企業で新卒採用人数を**7割削減**した例もある
- だからこそ、AI に仕事を奪われる側ではなく、**AI を使いこなす側**になるためのスキルが重要
- 本動画のテーマ：Codex を使って AI エージェントチームを作り、仕事を自動化して終わらせる方法の紹介

## 2. Codex とは

OpenAI（ChatGPT の開発元）が提供する、**エンジニア向けのコーディング AI エージェント**。

- Web サイトやアプリ開発に必要なコーディング作業を、会話するだけで作成・編集・修正してくれる
- コンセプトは **「Codex for almost everything」**＝ コーディングに限らず、ほぼあらゆる作業に対応
  - 例：大量の Excel データを 1 つの見やすい表にまとめる、といった事務作業も代行
- 文系・非エンジニアでも使えるように設計されている

### モデル（GPT 5.5）のポイント

- 賢くなったにもかかわらず、**トークン消費量が抑えられている**のが特徴
  - トークン＝ AI が情報処理に使う単位（日本語は概ね12文字で1トークン）
  - スマホの通信量制限のようなイメージ。制限を気にせず使い倒せる

### 画像生成も一体化

- ChatGPT と同じ OpenAI 製のため、**高品質な画像生成（GPT Image 2）**を標準搭載
- 画像生成 → そのままスライドやウェブサイト制作まで一気通貫で行える
- 事務作業だけでなく、クリエイティブ領域もカバー

## 3. Codex vs Claude Code

| 項目 | Codex | Claude Code |
|---|---|---|
| 開発元 | OpenAI | Anthropic |
| 通信制限 | Pro 以上のプランでも比較的余裕がある | Pro プランだとすぐ制限に達しやすい（Max プラン推奨） |
| 画像生成 | **標準搭載（GPT Image 2）** | 非対応 |
| 得意分野 | 事務作業＋クリエイティブを一括対応 | コーディング・デザイン領域 |

> なるべく通信制限にかからず使い続けたい人には Codex がおすすめ。

## 4. 始め方（料金プラン・ログイン）

### 料金

| プラン | 目安 | 内容 |
|---|---|---|
| 無料プラン | ¥0 | 使用制限あり |
| 有料プラン | 月額約3,000円 | ChatGPT（プラス）と Codex の両方が使える |

> まずはお試しでプラスプランから始めるのがおすすめ。

### ログイン〜インストール手順

```
1. Google で「ChatGPT」を検索してアクセス
2. メールアドレス or Google アカウント等で会員登録・ログイン
3. 左サイドバーの「Codex」をクリック
4. Codex を使える4つの環境から選択
   （ブラウザ／デスクトップアプリ／ターミナル／IDE）
5. 非エンジニアには「デスクトップアプリ」が最もおすすめ
   → ChatGPT と似た画面で馴染みやすい
6. デスクトップアプリをダウンロードして初期設定
```

## 5. 画面と設定の解説

### プロジェクト（フォルダ）選択

- 新しいフォルダを作って作業するか、既存フォルダを使うかを選択
- 選択したフォルダ名がチャット欄下部に表示される

### チャット欄の各設定

| 項目 | 内容 |
|---|---|
| モデル | GPT 5.5 など複数から選択可能 |
| インテリジェンス | AI がどれだけ深く考えるか（低〜非常に高い）。基本は「中」〜「高い」を推奨 |
| 速度 | 標準 or 高速 |

### 権限設定

| 設定 | 内容 |
|---|---|
| デフォルト権限 | ユーザーの確認をとりながら作業を進める（推奨） |
| 自動 / フルアクセス | 確認の手間は減るが、意図しないファイル削除等のリスクが増えるためオフのままが無難 |
| 構成の承認（Untrusted〜Never） | 承認を求めるタイミングの設定。安全志向なら Untrusted、ある程度任せるなら **On Request** が使いやすい |
| サンドボックス | 選択したフォルダ内でのみ作業させる **Workspace Write** を推奨 |

### チャット欄のその他機能

- **「＋」**：ファイルや写真の添付
- **プランモード**：作業前に計画を立ててから進めてくれるモード
- **プラグイン**：Google Drive、Gmail、Word など外部サービスと連携。以下3構成からなる
  - MCP 機能（外部連携）
  - スキル機能（特定作業の自動化）
  - アプリ連携

## 6. AIエージェントチームの作り方

### 組織図の考え方

```
プロジェクトリーダー（自分）
        ↓
   AIディレクター（メインエージェント／副リーダー）
        ↓
 サブエージェント（リサーチ／ライター／編集者／
 ビジュアル担当／SNS運用担当／営業担当 など）
```

- **メインエージェント**：ユーザーとやり取りしつつ、各サブエージェントに指示を出して全体を統括
- **サブエージェント**：役割に特化したAIが実行部隊として作業を進める
- MCP やスキルを組み合わせることで、Gmail の下書き作成なども自動実行可能

### AGENTS.md でチームのルールを作る

- `AGENTS.md` は Codex が最優先で確認する指示書ファイル（Claude Code では `CLAUDE.md`）
- 作業対象フォルダの直下にあらかじめ用意しておく
- 自分で1から作るのが難しければ、AI に候補を出してもらいながら埋めていく方法が簡単

> プロンプト例：「AIエージェントチームを運営するためにAGENTS.mdを作りたい。どのような項目や決まり事が必要か、考えられる候補を提示して」

## 7. 活用実演：記事執筆から営業資料まで

架空の設定：中小スポーツメーカーで、女性向けランニングメディアの運営を1人（＋AIチーム）で任された場合の実演。

### 実演1：リサーチ記事の執筆

1. 「梅雨でもランニングを続けたい女性向けの特集記事を書きたい」と指示
2. AIディレクター → リサーチ担当がWeb調査を実施
3. 提案企画が複数（3本）提示される
4. 企画を承認 → ライターが執筆 → 編集者がチェック・仕上げ → AIディレクターが最終確認
5. 参考文献・執筆者情報も明記された記事が完成

> 1人で行えば1日以上かかる作業が、指示と確認だけで完了。

### 実演2：SNS投稿コンテンツの作成

1. 「記事をSNS用に展開してほしい」と指示
2. ビジュアル担当・SNS運用担当が作業を分担
3. X／Instagram／note それぞれに最適化された投稿文・画像が生成される

> デザイン確認の無限ループ（依頼→確認→修正の繰り返し）を避けられる。ただし最終的な品質判断はプロにも確認してもらうのが望ましい。

### 実演3：タイアップ提案メール・LP・スライド作成

1. タイアップ先への提案メールの下書きを依頼 → 営業担当が作成（送信は手動確認）
2. 提案資料として物足りない場合は、特集を紹介するウェブサイト（LP）とプレゼンスライドの作成も依頼
3. 各サブエージェントが並行して作業し、LP・スライドが完成

> 1人なら同時並行が難しい「スライド構成・デザイン・文言」を、役割分担したAIチームに任せられる。

## 8. スマホによるリモート操作

- ChatGPT のスマホアプリから Codex を**遠隔操作**可能
- 外出先でもスマホから指示・確認ができ、作業が完結する
- 注意点：**PC 側で Codex を起動しておく必要がある**

## 9. まとめ・活用のポイント

- AI は日々進化しており、AI インフルエンサーや研究者ですら全情報を追い切れないと言われるほどのスピード
- 情報収集だけでなく、**実際に手を動かして試行錯誤する**ことが重要
- 各分野のスキルを持つ人が AI で自分の分身を作れる時代に入りつつあり、AI スキルの有無が今後の仕事の機会に直結する
- 日本企業でも新卒採用削減の動きが出ており、AI活用スキルの習得は急務

### 今回登場した主なポイントの整理

| テーマ | 要点 |
|---|---|
| Codex の強み | 事務作業＋画像生成まで一括対応、ChatGPTと同じ感覚で使える |
| チーム構成 | メインエージェント（AIディレクター）＋役割特化のサブエージェント |
| ルール作り | `AGENTS.md` に会社の決まり事を明文化 |
| 権限設定 | デフォルト権限＋On Request＋Workspace Write が安全な運用ライン |
| 活用範囲 | 記事執筆／SNS投稿／営業メール／LP・スライド作成まで一気通貫 |
| 外出先対応 | スマホからのリモート操作が可能（PC起動は必須） |
