## 1. Record & Replay とは

**OpenAI 公式が提供する Codex 向けプラグイン。**

> 自分がいつも行っている作業を「画面録画しながら手作業で実行」するだけで、  
> その動きを Codex が自動的に読み取り、  
> **全く同じ作業を再現するAIワークフロー（スキル）に変換してくれる**機能。

### 仕組みの流れ

```
① Record & Replay プラグインを起動 → 画面録画開始
② 録画中に「いつもの作業」を手作業で実行
③ 録画停止 → Codex が動きを解析
④ 自動的にスキル（AIワークフロー）を生成
⑤ 以降はスラッシュコマンド一言で同じ作業を全自動実行
```

### できること

- エンジニアでも完全な自動化を実現する強力なツール
- パソコン上でできる作業なら**ほぼ何でも自動化可能**
- 複雑な手順も動画を見せるだけで覚えてくれる

## 2. 必要な設定（2つのセットアップ）

### セットアップ① Record & Replay プラグインのインストール

1. 左上の「プラグイン」をクリック
2. 「record」で検索
3. **Record & Replay**（OpenAI 公式プラグイン）をインストール

### セットアップ② コンピューターユーズの有効化

Codex がマウスを動かして作業を**再現**するために必要。

1. 左下の「設定」をクリック
2. 「コンピューターの使用」をクリック
3. **「2位のアプリ」をオン**にする

> この2つをセットで設定することではじめて Record & Replay が機能する。

---

## 3. 基本的な使い方

### スキルの作成手順

1. プラグイン → Record & Replay →「チャットで試す」をクリック
   （デフォルトのシステムプロンプトが入力済みの状態になる）
2. 送信すると**画面録画が自動的に開始**
3. 録画中に「自動化したい作業」を手作業で実施
4. 録画を停止 → Codex がイベントログを解析 → スキルを自動生成
5. 必要に応じて**補足事項を追加入力**してスキルの精度を高める

### 生成されたスキルの確認

```
プラグイン → 管理 → スキル
```

スキル名（例：`create_number_folder`）が一覧に保存される。

### スキルの呼び出し方

スラッシュコマンドでスキル名を選択し、具体的な指示を添えて送信するだけ。

### 活用事例① 番号付きフォルダの自動作成

**シナリオ**：デスクトップに `001` 〜 `N` 番号のフォルダを作成し、右上に整列させる

**録画した手順**
1. デスクトップで新規フォルダを作成（001, 002, 003...）
2. 右クリック →「整理」で右上に綺麗に詰めて整列

**補足事項として追加したプロンプト**
```
デスクトップ上にフォルダを001番からユーザーが指定した数だけ作成する。
その後デスクトップの右上にフォルダを集めて、右クリック → 整理ボタンで
番号順に整列させる。コンピューターユーズを必ず使用すること。
```

**呼び出し例**
```
/create_number_folder 1から5まで作成して
```

**結果**：フォルダが自動生成され、右上に番号順で整列（成功）

### 活用事例② ウェブサイトのスクリーンショット一括収集

**シナリオ**：指定ジャンルの複数ウェブサイトを検索し、ページ全体のスクリーンショットをPDFで保存

**使用ツール**
- ブラウザ拡張機能「GoFullPage」（ページ全体を1枚でスクショ）
- Codex のコンピューターユーズ

**録画した手順**
1. ブラウザでウェブサイトを検索・表示
2. GoFullPage 拡張機能のボタンをクリック → PDF 保存
3. デスクトップ上の指定フォルダに格納
4. 複数サイト分繰り返す

**呼び出し例**
```
/web_screenshot_collector ガジェット関連のウェブサイトを5件取得してきて
```

**結果**
- ブラウザが自動で起動 → Google 検索 → 各サイトのスクショを自動取得
- 指定フォルダに PDF として格納（成功）

**活用場面**
- 競合サイト・参考デザインの一括収集
- 打ち合わせ用の参考資料まとめ
- 特定カテゴリの市場調査

### 活用事例③ X（Twitter）アナリティクスデータ収集

**シナリオ**：X のアナリティクスデータ（全体＋投稿別）を自動取得してフォルダにまとめる

**録画した手順**
1. X のアナリティクスページを開く
2. 指定期間（例：2週間）を選択
3. 全体データを CSV でダウンロード
4. 個々の投稿ページを開き、GoFullPage でスクリーンショット取得
5. 同一フォルダにまとめて保存

**ポイント**：X はログインが必要なため、ブラウザ操作型のコンピューターユーズが最適

> ログイン ID・パスワードを渡さなくても、自分の PC でログイン済みの状態なら Codex がそのまま操作できる。

**呼び出し例**
```
/x_analytics_collector 2週間分のデータをよろしく
```

**結果**
- X に自動アクセス → 2週間の期間選択 → CSV ダウンロード
- 各投稿のアナリティクス画面をスクリーンショット → フォルダに格納（成功）

### 活用事例④ ブログ記事の別サービスへの移行

**シナリオ**：Substack の記事を X（または他のブログサービス）に移行する

**課題**：直接コピペだと文字化けや画像が引き継がれない問題がある

**解決した手順（録画）**
1. 移行元（Substack）の記事から画像を**デスクトップのフォルダに一時保存**
2. 移行先（X）の新規記事作成ページを開く
3. テキストをコピペ
4. 保存した画像を**ファイルとして改めてアップロード**

> 一度デスクトップに保存 → 再アップロード のワンクッションが成功の鍵

**呼び出し例**
```
/article_migrate https://substack.com/xxx/my-article
```

**結果**
- 画像を自動保存 → X の記事作成ページに自動アップロード
- テキスト・画像ともに綺麗に移行（成功）

### 活用事例⑤ Notta 動画アーカイブの自動処理

**シナリオ**：コミュニティのセミナー動画を Notta にアップロードし、目次・共有リンク・文字起こしを一括作成

**毎週かかっていた手作業（10〜20分）**
1. 動画を Notta にアップロード
2. アーカイブから目次を生成
3. 目次をテキストエディタにコピペして編集
4. 共有リンクを発行（パスワード付き）
5. 文字起こしデータを CSV でダウンロード
6. 全データを同一フォルダに格納

**呼び出し例**
```
/notta_archive 動画ファイルをドラッグ&ドロップして送信
```

**結果**
- Notta が自動で起動 → 動画アップロード → 目次生成
- 共有リンク発行・パスワード設定 → テキストエディタにコピペ
- 文字起こしダウンロード → フォルダにまとめて格納（成功）

> 毎週の定例作業が**一言指示だけで完全自動化**

## 5. 成功させるコツ

### コツ①：録画だけで完結させない

動画の録画後に**補足テキストを必ず追加**する。

```
録画 → 停止 → 補足事項をテキストで追加入力 → 送信
```

補足に含めると良い内容：
- ユーザーが指定する変数（数、ジャンル、期間など）
- 具体的な保存先・ファイル形式
- 「コンピューターユーズを必ず使用すること」
- エッジケースへの対応方法

### コツ②：シンプルな繰り返し作業に向いている

コンピューターユーズは**単純・反復的な作業が最も安定**する。

| 向いている作業 | 向いていない作業 |
|---|---|
| ファイルの作成・整理 | 複雑な判断が必要な作業 |
| スクリーンショット取得 | リアルタイムの状況判断 |
| データのコピー・貼り付け | 予測不能な画面変化への対応 |
| ログイン済みサービスの操作 | 動的に変化する UI |

### コツ③：画像を含む移行作業は「一時保存」を挟む

コピペだと失敗しやすい場合は、**一度デスクトップに保存してから再アップロード**する手順を録画に含める。

## 6. まとめ

### Record & Replay で実現できること

```
「自分がやっている作業」= 「AIが代わりにやってくれる作業」
```

- プログラミング知識ゼロでも**複雑なAIワークフローを作成可能**
- 一度スキルを作れば**スラッシュコマンド一言**で何度でも再現
- 毎週・毎日の定例作業を完全自動化

### 向いている活用シーン

| カテゴリ | 具体例 |
|---|---|
| データ収集 | ウェブサイトのスクショ一括保存、SNS分析データ収集 |
| コンテンツ管理 | 記事の別サービスへの移行、動画アーカイブ整理 |
| ファイル整理 | フォルダ自動作成・整列、ドキュメントの分類 |
| レポート作成 | 定期データの取得・まとめ |

### 作業手順のまとめ

```
1. プラグイン → Record & Replay をインストール
2. 設定 → コンピューターユーズをオン
3. Record & Replay を起動 → 録画開始
4. いつもの作業を手作業で実行
5. 録画停止 → 補足テキストを追加
6. スキルが自動生成される
7. スラッシュコマンドで呼び出して活用
```
