# 各回フィードバックアンケート（毎回終了時実施）

__所要時間:__ 約3分
__実施タイミング:__ 各回の終了前（残り3分）
__実施方法:__ Googleフォーム等（全10回共通フォーマット。回ごとにフォームを複製するか、回の選択項目を設ける）
__目的:__ 各回の理解度・満足度をリアルタイムに把握し、次回以降の進行を調整する。全10回の推移データとして Before/After レポートにも活用する。

---

## フォーム設定

- __フォームタイトル:__ 第○○回 フィードバックアンケート
- __フォーム冒頭文:__ 「本日の研修お疲れ様でした。3分で終わる簡単なアンケートです。率直にご回答ください。」

---

## 運用ガイド（講師向け）

### 実施フロー

1. 各回の終了3分前にQRコード / URLをスライドに表示
2. 「最後に3分で今日のフィードバックをお願いします」と案内
3. 回答後、クロージング → 次回予告へ

### 集計・活用方法

| 指標 | 対応設問 | 活用方法 |
| --- | --- | --- |
| 理解度推移 | Q2 | 全10回の折れ線グラフ → 理解度の変化を可視化 |
| 業務活用イメージ | Q3 | 全10回の折れ線グラフ → 回を追うごとに活用イメージが具体化しているか |
| 進行スピード | Q4 | 次回以降のペース調整に即時反映 |
| 満足度推移 | Q5 | 全10回の折れ線グラフ → レポートの満足度推移に使用 |
| 自由記述 | Q6 | 次回冒頭で質問に回答 / 内容の補足 |

### レポートへの組み込み

Before/After 比較レポートに以下のグラフを掲載する：

- **理解度の推移グラフ**（Q2の平均スコア × 全10回）
- **業務活用イメージの推移グラフ**（Q3の平均スコア × 全10回）
- **各回満足度の推移グラフ**（Q5の平均スコア × 全10回）
- **進行スピードの適切性**（Q4の回ごとの分布）

### 注意事項

- 第01回は事前アンケート（5分）があるため、フィードバックアンケートは実施しない（時間が足りない場合）。ただし実施する場合は、クロージング後にオンラインで回答してもらう運用も可。
- 第10回は事後アンケート（5分）に統合されているため、フィードバックアンケートは不要。事後アンケートに満足度・理解度が含まれている。
- 実質的な運用対象は **第02回〜第09回**（計8回分）が標準。
