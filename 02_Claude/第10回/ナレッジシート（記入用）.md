## 研修後も使える：Cowork ソリューション設計フローチャート（再掲）

```
Q1. 解決したい業務の成果物は何か？
  ├─ 文書・メール・報告書 → 【文書生成タスク】
  ├─ Excel・集計表・グラフ → 【データ処理タスク】
  ├─ ファイルの整理・変換・リネーム → 【ファイル管理タスク】
  └─ 上記の組み合わせ → 【複合タスク】

Q2. その作業は定期的に繰り返すか？
  ├─ はい → 【スケジュールタスク】に設定
  └─ いいえ → 都度実行

Q3. 同じ前提条件を毎回使うか？
  ├─ はい → 【プロジェクト】にまとめる
  └─ いいえ → 単発タスクで十分
```
