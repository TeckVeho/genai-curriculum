# __第04回：Cowork入門 (2)：ファイル操作＋Excel・スプレッドシート自動生成__

## __1\. 講義概要__

- __フェーズ:__ 座学
- __所要時間:__ 60分
- __主要機能:__ Claude Desktop — Cowork（ファイル操作・Excel生成）
- __形式:__ オンライン

### __狙い__

前回の文書生成に続き、Cowork の**ファイル操作**と **Excel（スプレッドシート）自動生成**を体験する。「関数付きの Excel ファイルを日本語の指示だけで作れる」という成功体験を通じて、データ処理系の業務課題にも Cowork が使えるイメージを持ってもらう。ローカルファイルの読み込み → 加工 → 新規ファイル出力という一連の流れを体感することが最大のポイント。

### __到達目標__

1. Cowork でローカルファイルを読み込み、内容を分析・加工するタスクを実行できる
2. Excel ファイル（関数・書式付き）を Cowork で自動生成できる
3. 「読み込み → 加工 → 出力」の一連のデータ処理フローを理解している
4. 次回の複合タスク・プロジェクト活用に向けた見通しを持てる

### __準備物（講師）__

- プロジェクター・スクリーン（またはオンライン画面共有環境）
- 講師用PC（Claude Desktop の Cowork が動作する状態）
- 配布用サンプルデータ：**配送実績サンプル.csv**（受講者にダウンロードまたはチャットで配布）
- 事前に Cowork で作成したデモ用 Excel ファイル（完成版）

### __準備物（受講者）__

- PC（macOS または Windows）
- Claude Desktop アプリ（最新バージョン。Cowork の実行には Pro 以上が必要）
- 前回の棚卸しシート（手元に置いておく）

### __【05\-20分】 ハンズオン①：Excel ファイルをゼロから作る__

- __フェーズ:__ ハンズオン
- __スライド内容:__
	- __タイトル:__ Cowork で Excel を作ろう ── 関数・書式付き

- __講師スクリプト案:__  
「まず、何もないところから Excel ファイルを作ってもらいます。共通お題はこちらです。」

- __共通お題（全員同じタスクを実行）:__

> **Cowork へのタスク入力（コピーして使用）：**  
> 「デスクトップの『研修出力』フォルダに、以下の仕様で Excel ファイル `配送実績_サンプル.xlsx` を作ってください。  
> ■ シート名：4月実績  
> ■ 列構成：日付・配送先・品名・数量・単価・金額  
> ■ サンプルデータを5行入れてください（物流会社の配送を想定したリアルなデータ）  
> ■ 金額列は「数量×単価」の計算式にしてください  
> ■ 最終行に数量の合計（SUM関数）と金額の合計（SUM関数）を入れてください  
> ■ ヘッダー行に薄い青の背景色、合計行に薄い黄色の背景色をつけてください」

- __操作手順:__
	1. Claude Desktop の Cowork タブを開く
	2. 上記テキストを入力欄に貼り付けて送信
	3. Claude が計画 → 実行する過程を観察
	4. 完了後、「研修出力」フォルダを開いて `配送実績_サンプル.xlsx` を確認
	5. Excel で開き、①データ内容 ②計算式 ③書式 が期待通りか確認

- __解説ポイント:__
	- 「関数（SUM）が正しく入っているか、セルをクリックして確認してみてください」
	- 「書式（色・太字）まで指定通りに反映されていますね。これが Cowork の強みです」
	- 「イメージと違ったら、この後すぐに修正指示を出せます」

- __修正指示の練習（うまくいった人から）:__

> **修正指示例：**  
> 「数量列と金額列に3桁ごとのカンマ区切りを設定してください。また、金額列の合計セルを太字にしてください。」

### __【20\-40分】 ハンズオン②：既存ファイルを読み込んで加工する__

- __フェーズ:__ ハンズオン（メインパート）
- __スライド内容:__
	- __タイトル:__ 既存のデータを Cowork に読ませて加工しよう

- __講師スクリプト案:__  
「次は、**すでにあるファイルを Cowork に読み込ませて加工する**体験です。実際の業務では、既存の CSV や Excel を加工するケースの方が多いですよね。」

- __事前準備:__
	- 講師が配布する `配送実績_元データ.csv`（20行程度のサンプルデータ）を受講者がデスクトップの「研修出力」フォルダにダウンロード

- __共通お題:__

> **Cowork へのタスク入力：**  
> 「デスクトップの『研修出力』フォルダにある `配送実績_元データ.csv` を読み込んで、以下の加工を行い、結果を `配送実績_加工済み.xlsx` として同じフォルダに保存してください。  
> ■ 配送先ごとの合計金額を集計し、新しいシート「配送先別集計」に一覧表として作成  
> ■ 集計シートに配送先別の金額を棒グラフとして追加  
> ■ 元データのシートと集計シートの2シート構成にする  
> ■ ヘッダー行に書式（太字・背景色）を設定」

- __操作手順:__
	1. タスクを入力して送信
	2. Claude が CSV を読み込み → 分析 → Excel生成する過程を見守る
	3. 完了後、`配送実績_加工済み.xlsx` を開いて確認
	4. 元データシートと集計シートの2枚構成、グラフの有無を確認

- __解説ポイント:__
	- 「ローカルファイルのパスを伝えるだけで、Cowork が自分で読み込んでくれます」
	- 「CSV → 集計付きExcel という変換が、指示1回で完了しました。手作業でやったら何分かかるか想像してみてください」
	- 「グラフまで入った状態で出力されるので、そのまま上司に提出できるレベルです」

- __巡回指導のポイント:__
	- 「CSV が読み込めない」→ ファイルパスが正しいか確認。フォルダ名が日本語でも問題ないがスペルミスに注意
	- 「グラフが入っていない」→ 「棒グラフを追加してください」と修正指示を出す
	- 「集計が合わない」→ 元データの金額列の形式を確認。修正指示で「金額列を数値として扱ってください」

## __3\. 想定される質問と回答 \(Q&A\)__

- __Q1: 「Cowork が作った Excel の関数が間違っていたらどうしますか？」__
	- __A:__ 「実行後にファイルを開いて確認し、間違いがあれば『SUM関数の範囲が違います。B2:B6に修正してください』のように具体的に修正指示を出してください。Claude は自分の出力を自分で直すのが得意です。」

- __Q2: 「大きなデータ（数千行）でも処理できますか？」__
	- __A:__ 「基本的に対応できます。ただし、非常に大きなデータの場合は処理時間が長くなることがあります。研修で扱う規模（数十〜数百行）では問題ありません。」

- __Q3: 「Excel 以外のファイル形式も作れますか？」__
	- __A:__ 「はい。Word文書、PowerPointプレゼンテーション、CSV、テキストファイル、PDFなど、さまざまな形式に対応しています。第06回以降のワークショップで、自分の業務に合った形式を選んで作ります。」

- __Q4: 「自分のPCにある機密データを Cowork に読ませても大丈夫ですか？」__
	- __A:__ 「Cowork はローカルで実行されますが、ネットワークアクセスが発生する場合があります。また Research Preview のため監査ログの対象外です。機密データ（個人情報・顧客情報・社外秘の金額等）は、マスキングしてから使うか、ダミーデータで代替してください。これは研修後の実務でも重要なルールです。」
