## __1\. 講義概要__

- __フェーズ:__ 応用 / Phase 1 締め
- __所要時間:__ 60分
- __主要機能:__ ディープリサーチ・エージェントモード・全機能統合
- __形式:__ オンライン

### __狙い__

ChatGPT の最高レベルの2機能——**ディープリサーチ（詳細調査レポート自動生成）** と **エージェントモード（複数ステップの自律実行）** ——を体験し、「調べる・まとめる・作る」を一気通貫で ChatGPT に任せられることを体感する。さらに Phase 1（第01〜06回）で学んだ全機能を組み合わせた **統合ケーススタディ**を行い、Phase 2（実践ワークショップ）への橋渡しとする。

### __到達目標__

1. ディープリサーチを起動して、業務に関連する調査レポートを生成できる
2. エージェントモードで複数ステップのタスクを丸投げし、自律実行の様子を観察できる
3. ChatGPT の全機能の使い分け（どの業務にどの機能を使うか）を一枚の図で整理できる
4. Phase 2 で取り組む業務テーマを決め、必要な事前準備を理解している

### __準備物（講師）__

- 講師用 PC（ChatGPT Plus 以上——ディープリサーチ・エージェントモードは Plus 以上）
- デモ用の調査テーマ（例：「物流業の2024年問題と中小企業の対策」）
- 「ChatGPT 機能選定マトリックス」スライド（本テキスト4節参照）
- Phase 2 ブリッジ用「AI業務設計フォーマット」（第07回配布予定のプレビュー）

### __準備物（受講者）__

- PC（ChatGPT ログイン済み）
- 第05回の宿題（カスタムGPT / プロジェクトを試した感想・メモ）
- 「Phase 2 で改善したい業務を1つ」頭の中で考えておく

#### __ディープリサーチ デモ（12分）__

**起動方法：**

1. ChatGPT の入力欄下部「ディープリサーチ」ボタンをクリック（または「新しいチャット」から選択）
2. プロンプト入力：

> 「物流業界の2024年問題（ドライバーの時間外労働規制）について、中小企業の物流会社が取るべき事務・人事面の対策を、公的機関や業界団体の情報を中心に調査して、A4 2〜3枚程度の調査レポートを作成してください。見出し・箇条書き・出典URL付きで。」

3. 調査中の進行状況（参照中のサイト一覧）をリアルタイムで投影
4. 完成したレポートを確認：
	- 見出し構成・箇条書きの整理
	- 出典 URL の一覧
	- 「参照しました：〇〇サイト ×× 件」の表示

5. 続けて：

> 「このレポートの内容を、社内で上司に口頭説明するための5分のプレゼンメモに変換してください。」

→ ディープリサーチの結果を素材に、さらに文書を派生させる使い方を見せる

- __注意点:__
	- 内容はハルシネーションのチェックが必要。出典 URL を必ず1つ開いて確認する
	- Plus プラン以上の機能（無料版では利用不可の場合あり）

- __講師スクリプト案:__  
「**エージェントモード**は、ChatGPT が**タスクを自分で分解して、複数ステップを自律的に実行する**機能です。  

今まで『1ステップずつ指示する』必要があった複数工程の作業を、**一度の指示で全部やってもらえます。**  

例えば：  
- 『〇〇について調べて → 重要ポイントを箇条書きにして → 上司向けの説明メモを A4一枚で作って』  
- これを1つのプロンプトで投げると、エージェントが自分でステップを組み立てて実行します。  

**ディープリサーチとの違い：** ディープリサーチは「調査→レポート作成」に特化。エージェントモードは「調査・集計・文書作成・ファイル操作」など、より広い種類のタスクを組み合わせて実行できます。」

#### __エージェントモード デモ（12分）__

**起動方法：**

1. ChatGPT 左サイドバーの「エージェント」または、新規チャットで「o3」モデルを選択
2. プロンプト入力：

> 「以下の作業を順番に実行してください。
> ①物流業界のインボイス制度対応で、中小企業の事務担当が2025年に注意すべき最新情報を調査する。
> ②調査結果から、対応チェックリスト（5〜7項目）を作成する。
> ③そのチェックリストをもとに、社内の一般事務スタッフ向けの『インボイス対応確認シート（A4一枚）』のたたき台を作成する。
> 各ステップの出典 URL も記載してください。」

3. エージェントが**ステップを自律的に計画**する様子を投影（「Step 1: 検索中…」「Step 2: 整理中…」「Step 3: 文書作成中…」）

4. 完成した確認シートを確認し、講師コメント：
	- 「今まで30分かかっていた作業が、指示を出して待つだけで完成しました」
	- 「ただし、必ず内容の検品（数字・事実確認）をしてください」

5. さらに追加指示：

> 「この確認シートをキャンバスで開いて、もう少し現場向けにわかりやすくしてください。」

→ エージェントモードで生成した結果を、キャンバスで仕上げる**機能の連携**を見せる

## __3\. 想定される質問と回答 \(Q&A\)__

- __Q1: 「ディープリサーチとエージェントモード、どちらを使えばいいですか？」__
	- __A（回答）:__ 「目的によって使い分けます。**『調査・情報収集→レポート作成』に特化したい → ディープリサーチ**。**『調査だけでなく、集計・文書作成・ファイル操作まで複数の種類のタスクを組み合わせたい → エージェントモード』**。迷ったらディープリサーチから試してください。より速くシンプルに使えます。」

- __Q2: 「エージェントモードが途中で止まることがあります。」__
	- __A（回答）:__ 「稀に途中で停止することがあります。**『続きから続けてください』と入力**するか、プロンプトを少し具体化して再試行してください。または、複雑すぎるタスクを2〜3に分けて別々に依頼すると安定することが多いです。」

- __Q3: 「機能が多すぎて、毎回どれを使うか迷います。」__
	- __A（回答）:__ 「今日の**機能選定マトリックス**を印刷して、デスクに貼っておいてください。最初は『まずテキスト生成（通常のチャット）から試す。うまくいかない・もっと良くしたい場合に別の機能を試す』というルールでOKです。全機能を毎回考える必要はありません。」

- __Q4: 「Phase 2 の宿題、機密情報のマスキングはどうすればいいですか？」__
	- __A（回答）:__ 「別紙の **『AI入力OK_NGチェックリスト』** を参照してください。基本的なマスキングの考え方は：顧客名→『A社』、金額→おおよその桁数に変更か削除、電話番号→削除、個人名→イニシャル or『担当者』に変更。チェックリストに沿って確認してから試してください。」
