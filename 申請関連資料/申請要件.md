# 人材開発支援助成金「事業展開等リスキリング支援コース」申請要件チェックリスト

要件を「提供側」「受講側」「両側（共通）」に分けてまとめます。

---

## ✅ 提供側（教育訓練機関・セミナー運営会社）の要件

- [ ] **法人格と事業目的**
  日本国内の法人であり、定款または登記簿の事業目的に**「教育訓練事業」等の記載があること**が必須です。海外法人によるセミナーは対象外となります。

- [ ] **情報の公開義務**
  eラーニングや定額制サービスの場合、**自社HP等で訓練の概要や申込方法を広く一般に公開**している必要があります。「特定の事業主限定」の裏メニューや、SNS・メール案内のみの密室的な訓練は不可です。

- [ ] **不適切な勧誘・キックバックの禁止**
  **「実質無料」「支払額以上の利益」といった謳い文句での勧誘は禁止**されています。また、アンケート協力金や営業協力金等の名目で受講側にキックバックを行うことは「実質的な経費の減額（返金）」とみなされ、経費助成の対象外（最悪の場合は不正受給）となります。

- [ ] **審査への協力と透明性**
  支給申請承諾書（様式第12号）へ虚偽なく署名し、労働局の審査や実地調査に協力する体制が必要です。また、営業時に顧客へ配布した提案書やチラシ等の資料一式を労働局へ提出できる透明性が求められます。

- [ ] **厳格な受講管理**
  出席簿やLMS（学習管理システム）のログを用いて、分単位で受講状況を管理・提示できることが求められます。

---

## ✅ 受講側（申請事業主・労働者）の要件

- [ ] **社内体制の整備と計画周知**
  計画届の提出日までに**「職業能力開発推進者」を選任**し、会社としての**「事業内職業能力開発計画」を策定のうえ全従業員へ周知**することが必須要件です。

- [ ] **経費の全額負担の原則**
  支給申請日までに、**事業主が受講料等の経費を「会社名義」で全額負担（振込）**している必要があります。従業員に一部でも負担させたり、従業員による立て替えやポイント払いを行ったりした場合は不支給となります。

- [ ] **賃金の適正な支払いと記録**
  タイムカード等で訓練時間と休憩時間・移動時間を明確に区別して1分単位で記録し、**訓練時間に対して適正な賃金（所定外や休日の場合は割増賃金）を支払う**必要があります。eラーニングも業務時間内に実施し、その分の賃金支払いが必須です。

- [ ] **手続期限と変更届の厳守**
  訓練開始日の1か月前までに計画届を提出すること、そして予定に変更（日程、講師、対象者の追加など）がある場合は**必ず事前の「変更届」を提出**することが義務付けられています。

- [ ] **その他の手続き要件**
  雇用保険適用事業所であり対象者が被保険者であること。業務命令による受講について労働者から承諾書を得ること。また、中小企業の場合は事前に「認定経営革新等支援機関」の確認印を得る必要があります。

---

## ✅ 両側（共通）の要件

- [ ] **訓練内容の適格性**
  訓練内容が**「事業展開」「DX（デジタル化）」「GX（グリーン化）」、または「人事計画に基づく新たな職務への転換」**のいずれかに該当する専門的な知識・技能の習得であること。既存アプリの初歩的な操作説明、汎用スキル（マナー研修等）、法定義務のある講習などは対象外です。

- [ ] **実訓練時間数（10時間以上）**
  休憩や移動時間を除いた**「実訓練時間」が10時間以上**であることが絶対条件です。1分でも不足するとコース全体が不支給となります。定額制（サブスクリプション）の場合も、対象労働者の合計受講時間が10時間以上である必要があります。

- [ ] **出席率と修了要件**
  通学制等の場合は**実訓練時間の8割以上に出席**すること。eラーニング等の場合は訓練期間中にすべてのカリキュラムを修了することが必要です。

- [ ] **広告宣伝費等の授受禁止**
  訓練成果のレビューやインタビューに協力する対価として、教育訓練機関から受講側へ金銭を支払う（受け取る）ことは、実質的な受講料の返金とみなされ禁止されています。
