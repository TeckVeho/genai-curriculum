# 受講企業向け 助成金申請 Todoリスト
## 人材開発支援助成金「事業展開等リスキリング支援コース」

---

## ⏰ 期限（厳守・1日でも遅れると不可）

| フェーズ | 期限 |
|---------|------|
| 計画届の提出 | **訓練開始の1か月前まで** |
| 変更が生じた場合 | **変更前日まで**に変更届 |
| 支給申請 | **訓練終了翌日から2か月以内** |

---

## ✅ フェーズ1：計画段階（訓練開始の1か月前まで）

### 社内体制の整備

- [ ] 職業能力開発推進者を選任する（課長級以上が望ましい）
- [ ] 事業内職業能力開発計画を策定し、全従業員に周知する（掲示・メール等）
- [ ] 対象労働者から業務命令による受講の承諾書を取得する
- [ ] （中小企業のみ）認定経営革新等支援機関の確認印を受ける

### 書類の作成

- [ ] [様式第1-1号](https://view.officeapps.live.com/op/view.aspx?src=https%3A%2F%2Fwww.mhlw.go.jp%2Fcontent%2F11800000%2F001604507.xlsx&wdOrigin=BROWSELINK)：職業訓練実施計画届
- [ ] [様式第1-3号](https://view.officeapps.live.com/op/view.aspx?src=https%3A%2F%2Fwww.mhlw.go.jp%2Fcontent%2F11800000%2F001666063.xlsx&wdOrigin=BROWSELINK)：事業展開等実施計画
- [ ] [様式第3-1号](https://view.officeapps.live.com/op/view.aspx?src=https%3A%2F%2Fwww.mhlw.go.jp%2Fcontent%2F11800000%2F001468820.xlsx&wdOrigin=BROWSELINK)：対象労働者一覧（定額制は第3-2号）
- [ ] [様式第11号](https://view.officeapps.live.com/op/view.aspx?src=https%3A%2F%2Fwww.mhlw.go.jp%2Fcontent%2F11800000%2F001666072.xlsx&wdOrigin=BROWSELINK)：事前確認書

### 管轄労働局（またはハローワーク）へ提出

- [ ] 上記書類＋運営側から受け取った資料（カリキュラム・契約書等）を一括提出する
  - 郵送の場合は**消印不可・到達日**が基準

---

## ✅ フェーズ2：訓練実施中

- [ ] 変更（日程・講師・場所・対象者等）が生じたら**前日まで**に変更届を提出する
- [ ] タイムカード・出勤簿で始業・終業時刻を**1分単位**で記録する
- [ ] 訓練時間と休憩・移動時間を**明確に分けて**記録する
- [ ] 各受講者の出席率が実訓練時間の**8割以上**あることを確認する
- [ ] 受講料を**会社名義の口座から全額振込**する（従業員立替・ポイント払いは不可）
- [ ] 訓練時間分の賃金を適正に支払い、記録する

---

## ✅ フェーズ3：支給申請（訓練終了翌日から2か月以内）

### 自社で作成する書類

- [ ] [様式第4-2号](https://view.officeapps.live.com/op/view.aspx?src=https%3A%2F%2Fwww.mhlw.go.jp%2Fcontent%2F11800000%2F001517881.xlsx&wdOrigin=BROWSELINK)：支給申請書
- [ ] [共通要領様式第1号](https://view.officeapps.live.com/op/view.aspx?src=https%3A%2F%2Fwww.mhlw.go.jp%2Fcontent%2F11600000%2F001470657.docx&wdOrigin=BROWSELINK)：支給要件確認申立書
- [ ] [様式第13号](https://view.officeapps.live.com/op/view.aspx?src=https%3A%2F%2Fwww.mhlw.go.jp%2Fcontent%2F11800000%2F001468842.xlsx&wdOrigin=BROWSELINK)：事業所確認票
- [ ] [様式第5号](https://view.officeapps.live.com/op/view.aspx?src=https%3A%2F%2Fwww.mhlw.go.jp%2Fcontent%2F11800000%2F001666074.xlsx&wdOrigin=BROWSELINK)：賃金助成の内訳
- [ ] [様式第6-2号](https://view.officeapps.live.com/op/view.aspx?src=https%3A%2F%2Fwww.mhlw.go.jp%2Fcontent%2F11800000%2F001666075.xlsx&wdOrigin=BROWSELINK)：経費助成の内訳
- [ ] 支払方法・受取人住所届（口座未登録の場合のみ）

### 自社で準備する証拠書類

- [ ] 雇用契約書または労働条件通知書の写し
- [ ] 賃金台帳または給与明細書の写し（訓練期間中のもの）
- [ ] 出勤簿またはタイムカードの写し
- [ ] 受講料の請求書・領収書の写し（または振込通知書等）

### 管轄労働局へ提出

- [ ] 上記書類＋運営側から受け取った書類（報告書・修了証・承諾書等）を一括提出する
