# __第02回：プロンプト基礎＋Web検索リサーチ体験：AIを動かす「言葉の技術」__

---

## __1\. 講義概要__

- __フェーズ:__ 基礎
- __所要時間:__ 60分
- __主要機能:__ テキスト生成・Web検索機能
- __形式:__ オンライン

### __狙い__

「AIが使えない」のではなく、**「指示（プロンプト）が曖昧」**なのが原因だと気づき、ChatGPT から100点満点の回答を引き出すための「指示書の書き方」をマスターする。後半では **ChatGPT の Web 検索機能** を使ったリサーチ体験を通じ、「調べ物も ChatGPT 1つで完結できる」ことを実感する。

### __到達目標__

1. プロンプトの「4要素（役割・状況・制約・形式）」を使って、業務に使えるメール・文書の下書きを生成できる
2. 曖昧な指示と具体的な指示の違いを自分の言葉で説明できる
3. ChatGPT の Web 検索機能をオンにして、最新情報を引用付きで取得できる
4. 検索結果の「引用（出典）」を確認し、情報の正確性を検品できる

### __準備物（講師）__

- 講師用 PC（ChatGPT ブラウザ版）
- 配布用ワークシート：「プロンプト修正ワークシート」（悪い指示→良い指示の変換練習）
- 前回宿題の共有用チャット（Slack / Google Chat 等）の URL を表示

### __準備物（受講者）__

- PC（ChatGPT アカウントログイン済み）
- 前回の宿題（ChatGPT に何かを聞いた体験）を口頭か チャットに準備しておく

#### __4要素の解説（5分）__

- __講師スクリプト案:__  
「AI への指示（これを**プロンプト**と呼びます）に必要なのは、次の4つだけです。事務の仕事に置き換えて説明しますね。

**1つ目：役割（Role）**  
AI にどんな帽子を被せるかです。『経理のベテランとして』『謝罪メールのプロとして』と指定します。役割を与えると、AI はその立場から回答してくれます。  

**2つ目：状況（Context）**  
これが一番大事。『納品が1日遅れた』『請求書の金額に誤りがあり、取引先がお怒りだ』という背景情報です。同僚に状況説明するのと同じ感覚です。背景が詳しいほど、AI の回答が的確になります。  

**3つ目：制約条件（Constraint）**  
『300文字以内で』『です・ます調で』『箇条書きで』というルールです。書式や社内ルールみたいなものです。  

**4つ目：出力形式（Output）**  
メールで作るのか、表にするのか。荷物の荷姿指定（パレットかバラか）と同じです。  

この4つを揃えるだけで、AI の動きは劇的に変わります。」

- __スライド補足（4要素テンプレート）:__

```
#役割
あなたは [物流会社の一般事務担当 / ベテランの経理担当] です。

#状況（コンテキスト）
[取引先A社への納品が1日遅れた / 請求書の金額に誤りがあった] という状況で、
[謝罪と再発防止を伝えるメール / お詫びと訂正書類の送付案内] を作成してください。

#制約条件
・文字数は [200文字程度 / 300文字以内]
・トーンは [誠実に / 丁寧に / 簡潔に]
・[専門用語は使わない / 箇条書きを使う]

#出力形式
[メール本文として出力 / 箇条書きリストで / 表形式で]
```

## __3\. 想定される質問と回答 \(Q&A\)__

- __Q1: 「いちいち細かく書くのが面倒です。自分で書いた方が早くないですか？」__
	- __A（回答）:__ 「おっしゃる通り、最初はそう感じるかもしれません。でも、**よく使う指示テンプレートはメモ帳や ChatGPT のプロジェクト機能に保存**しておけばいいんです。『お詫びメールセット』『日報セット』みたいなテンプレートを一度作れば、あとは状況を少し書き換えるだけ。30分かかっていたメール作成が3分で終わります。初期投資だと思って、今日の演習で1つ作っておきましょう。」

- __Q2: 「思った通りの答えが出ない時、どうすればいいですか？」__
	- __A（回答）:__ 「一発で正解を出そうとしなくて大丈夫です！ **チャットを続けて改善**していくのが ChatGPT の使い方の基本です。『もっと短く』『丁寧な言葉に変えて』『箇条書きにして』と追加指示を出せば、どんどん修正してくれます。熟練の事務員に何度か赤入れするイメージです。」

- __Q3: 「Web 検索の情報、全部信用していいですか？」__
	- __A（回答）:__ 「**信用してはいけません（笑）。**ChatGPT が検索して要約してくれますが、要約が不正確な場合があります。特に法律や税制などの重要な情報は、**必ず引用リンクをクリックして原文（国税庁・厚生労働省等の公式サイト）を確認**してください。AIは『最初の入口』として使い、最終確認は公式情報で行うのが鉄則です。」

- __Q4: 「同じプロンプトを毎回入力するのが面倒です。保存できますか？」__
	- __A（回答）:__ 「いくつか方法があります。①**PC のメモ帳やドキュメントにコピペ用テンプレートを保存**する。②**ChatGPT の『プロジェクト』機能**（第05回で学びます）に定型プロンプトを保存する。③**キーボードの辞書登録**（例：『/おわび』で長いプロンプトが出てくるよう登録）。今日は①のメモ帳保存を試してみてください。」

### __AIへの指示は、4要素を埋めるだけで劇的に良くなります！__

```
#役割
あなたは [物流会社の事務担当 / ベテラン経理担当 / 謝罪メールのプロ] です。

#状況（コンテキスト）
[A社への納品が1日遅れた / 請求書の金額に誤りがあった / 新人に経費精算を説明したい]

#制約条件
・文字数：[200文字程度 / 300文字以内 / A4一枚程度]
・トーン：[誠実・丁寧 / 親しみやすく / 簡潔に]
・その他：[専門用語を使わない / 箇条書きを使う / 件名も考えてください]

#出力形式
[メール本文として / 箇条書きで / 表形式で / メール本文＋件名候補3つ]
```

### __【次回までの宿題】__

自分の業務を1つ選んで、4要素テンプレートを使って ChatGPT に試してみましょう！

- **テキスト生成（業務系）：**「いつものメール or 日報」を4要素で指示して生成する
- **Web 検索：** 業務に関係する法改正・業界ニュースを1つ ChatGPT の Web 検索で調べてみる
- ※うまくいかなくてもOK！「どんな指示だと失敗するか」を知るのも勉強です。
- ※次回冒頭、チャット等で一言シェアします。

**★ 次回予告（第03回）：**「えっ、手書きが30秒でデータに？」——ChatGPT に「目」をつける**マルチモーダル（画像読取・画像生成）**と、データを自動分析する**コードインタープリター**をお届けします！
